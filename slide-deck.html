2 · Problem Insight
Who has the problem — and what we learned

250+ KFC kiosks show everyone the same menu

  • The “you may also like” box is hand-picked by marketing, once a month — identical for an office tower at noon and a mall on Saturday night.
  • KFC’s own estimate: 15–20% of order value is left on the table (target: +10–15% AOV).

Learning #1 — from the data: 250 stores are not one store. Basket patterns split hard by site type (mall / office / residential / tourist), hour, holiday and live inventory.

Learning #2 — from psychology: kiosk upsell fails when it feels like a slot machine. Customers are time-pressed; trust is the only durable growth lever — so the agent saves them money before it sells.

Sources: KFC Vietnam track brief · menu-engineering & QSR-margin literature · 9,000-basket POS simulation
3 · Agentic Workflow
Not a chatbot — it senses, decides and acts on its own

Goal → Plan → Tools → Act → Verify

🎯 Goal

Raise this session’s order value without breaking trust. Triggered by the session itself — no prompt, no chat.

session_start → agent wakes

🧠 Plan

Builds a customer hypothesis (camera glance + every tap + live cart) and picks a strategy per slot: cross-subsidy? swap? decoy? complete-the-meal? or stop selling.

vision + 8B refine · 8-signal scorer

🔧 Tools

Chooses its own data & tools: cluster POS pairs, store inventory, promo calendar, profile store, gpt-oss pitch writer; ordering agent wields 9 D1-grounded tools.

D1 batch ~70ms · Workers AI

⚡ Act

Acts on every step of the native journey: re-ranked grid + best-seller tags, poster deal popup, savings flag, drink-led ADD ON, money-saving swap, last-chance chip, comeback voucher, festive re-skin. Places orders, hands off to humans.

preemptive: computed before the screen opens

✅ Verify

Logs impression → accept → order; computes an honest AOV counterfactual. LLM slow? Falls back to deterministic copy. Never suggests what a store can’t serve.

self-checks + live telemetry

Decisions the agent makes by itself: which psychology play per slotwhich deal to pop as a posteroffer the cheaper swap firststop selling when the meal is completerest an item after 2 unanswered offerspush overstock / protect stockoutsescalate to a human

Every decision streams to a live system diagram — the judges can watch it think
